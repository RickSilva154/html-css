Como a internet funciona?

Protocolos

Protocolos são uma linguagem ou idiomas unicos que todos podem usar,
facilitando a conexão de varias maquinas em um site

TCP/IP

IP identifica onde está o servidor ou cliente e o TCP divide o arquivo
que o cliente solicita e envia por pacotes, em rotas ou caminhos

WWW

É uma subrede da internet, também um conjunto de servidores especializados em HTTP.

BIT

São sinais eletricos que fazer um computador entender o que cada funçao faz.
A computação usa um conjunto de 8bits, que é a porção minima para se representar
algum dado, a cada 8bits é formado 1byte e nao se pode esquecer que:
MegaBytes= Armazenamente e Megabites= Transmissão

MODEM

O aparelho que faz modulação e demodulação

SERVIDOR DNS

É uma "agenda eletronica" da internet que salva os endereços IP dos servidores
e leva os clientes ate ele

DOMININO

Nome que o dns usa para identificar um IP como um site

HOSPEDAGEM

Local onde o site fica armazenado
